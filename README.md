#UFW Firewall Configuration 

This project sets up a basic firewall using UFW on kali linux 

##Steps 
1.Install UFW using apt.
2.Enabled UFW.
3.Allowed SSH Traffic (port 22)
4.Denied HTTP Traffic (port80)
5.Verified rules with 'ufw rules'.

## Deliverables
- ufw_configuration.sh Script with commands.
- Screenshot : Shows firewall rules active.
- README.md :Explains configuration.
